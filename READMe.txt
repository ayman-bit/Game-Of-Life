To run the GAME OF LIFE on your machine you have to do the following: 

1- Compile the code by typing the following in the terminal of the root directory: javac Sim247.java

2- Run the game by typing the following command: java Life [# of iterations] [Type of layout] 

Notes: 

	1- # of iterations can be whatever you desire
	2- The size of the grid can also be whatever you desire
	3- the existing layout types are as follows: 
		1- (M) for Mazing Oscillator
		2- (T) for Turtle glider
		3- (R) for Random
